# Al-Quds Educational Academy

A responsive static educational website built with **HTML**, **CSS**, and **JavaScript** to present the academy’s services, courses, teachers, and student testimonials through a clean Arabic interface.

> 🌐 Live Demo: https://al-quds-academy.netlify.app/
## Overview

This project is designed to showcase the identity and educational offerings of **Al-Quds Educational Academy** in a simple and user-friendly way. It features a fully Arabic layout, organized course sections, teacher profiles, student reviews, and direct WhatsApp contact links.

## Pages

- `index.html` — Home page
- `cours.html` — Courses page

## Features

- Responsive Arabic user interface
- Consistent visual identity with a Palestinian-inspired theme
- Organized courses section with clear lesson cards
- Teachers section
- Student testimonials section
- Direct WhatsApp contact links
- Simple and clean static website structure

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## Run Locally

1. Download or clone the project
2. Open `index.html` directly in your browser

## Project Purpose

The purpose of this website is to provide a clear and attractive online presence for the academy, making it easier for students and parents to explore available courses, learn about the teachers, and contact the academy quickly.
